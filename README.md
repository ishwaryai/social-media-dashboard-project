# Social Media Dashboard Starter

A simple Social Media Dashboard with:

Backend: Node.js + Express + PostgreSQL

Frontend: React + Vite + Recharts

1. Backend

cd backend
cp .env.example .env
npm install
npm run dev

2. Frontend

cd ../frontend
cp .env.example .env
npm install
npm run dev
